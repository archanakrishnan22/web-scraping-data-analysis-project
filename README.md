# Web Scraping & Data Analysis: Jumia Products

## Overview
This project involves scraping product data from Jumia's e-commerce platform and analyzing key attributes such as product names, prices, and ratings.

## Objective
To collect structured data from an online platform and convert it into a usable dataset for analysis.

## Tools Used
- Python
- BeautifulSoup
- Requests
- Pandas

## Key Features
- Scraped product data across multiple pages
- Extracted product name, price, and ratings
- Converted raw HTML data into structured dataset
- Exported data to CSV for further analysis

## Output
- products_from_jumia.csv
